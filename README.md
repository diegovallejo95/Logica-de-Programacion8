LOGICA DE PROGRAMACION SEMANA 8

NOMBRE DEL PROYECTO:
Juego interactivo de Piedra, papel y tijeras

INTEGRANTES:
Diego Vallejo

OBJETIVO DEL SISTEMA: 
Desarrollar un programa que permita al usuario jugar una partida de Piedra, Papel o Tijera contra la computadora, aplicando estructuras básicas de programación como variables, condicionales, ciclos y generación de números aleatorios.

DESCRIPCION DE FUNCIONALIDADES:

El sistema permite:

- Mostrar un mensaje de bienvenida al usuario.
- Permitir que el jugador seleccione una opción:
  - 1: Piedra
  - 2: Papel
  - 3: Tijera
- Generar una elección aleatoria para la computadora.
- Validar que la opción ingresada por el usuario sea correcta.
- Comparar la elección del jugador con la computadora.
- Determinar el resultado del juego:
  - Empate.
  - Victoria del jugador.
  - Victoria de la computadora.
- Mostrar en pantalla las elecciones realizadas y el resultado final.
